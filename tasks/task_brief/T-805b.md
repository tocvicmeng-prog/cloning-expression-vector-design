# T-805b Task Brief — `app.sop_protocol_orchestrator` (v1.2 — renamed + scoped per B2-09)

## Scope

Implement `app.sop_protocol_orchestrator` as the post-authorisation application service for operational SOP bundles. The orchestrator dispatches to `engine.sop_protocol` only after an `OperationalProtocolAuthorised` design event is observed for the current session, produces deterministic `SopProtocolBundle` payloads carrying rendered SOP artefact hashes and authorisation evidence, and emits `SopRendered` to the design stream.
