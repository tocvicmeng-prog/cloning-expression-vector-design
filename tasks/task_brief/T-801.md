# T-801 Task Brief — `engine.risk_classification` (B3 mitigation)

## Scope

Implement the Phase 8a B3 mitigation advisory-report generator for `engine.risk_classification`.

- Parse validated `catalogues/risk_advisories.yaml` payloads into a deterministic advisory catalogue.
- Generate `RiskAdvisoryReport` values carrying v1.5 identity fields, advisory catalogue version/hash, stable advisory IDs, and deterministic report content hashes.
- Infer trigger tags from biosafety tier, cargo/vector/host/intended-use metadata, screening verdicts, export/provenance markers, and AI-evasion watch metadata.
- Preserve active-advisory semantics by emitting caution and strong-caution advisories through the domain `RiskAdvisoryReport.required_acknowledgements()` surface.
- Map each emitted advisory to a graded citation from the catalogue payload.
