# T-804 Task Brief — `engine.controls`

## Scope

Implement the Phase 8a first-class control-set generator for `engine.controls`.

- Generate positive, negative, process, vehicle/mock, and library-specific `ControlDesign` values as a `ControlSet`.
- Tie generated controls to host role, assembly chemistry, cargo class, vector/delivery class, intended readout, library size, and replicate recommendation.
- Validate control sets for positive-control suitability, negative-control absence-of-signal, vehicle/mock controls, library-specific controls, process controls, and replicate structure.
- Keep the module non-operational and independent of gated protocol types.
