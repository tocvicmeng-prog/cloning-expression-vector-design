# T-314b Task Brief — Production profile + decision-record signing adapters

## Scope

Implement the production half of the T-314 signing split from CODING_AGENDA § 2.3.17:

- Ed25519 institutional `AuthorisationProfileSigner` and verifier.
- Ed25519 per-principal `DecisionRecordSigner` and verifier.
- Shared signing-key archive with current-key pointers, historical verification, rotation, and revocation.
- `DecisionRecordKeyManagementService` for provisioning, rotation, revocation, and governance public-key events.
- Offline verifier tools for authorisation profiles and signed decision records.
- Runbooks for profile-signing and decision-record-signing lifecycle operations.
