# T-1201 Task Brief — `app.constraint_translator` + `adapter.llm`

## Scope

Implemented the Phase 12 free-text translator and LLM adapter boundary:

- `src/app/constraint_translator.py` now defines typed translation proposals,
  structured constraints, `AdvisoryTextPolicy`, forbidden-output detection,
  citation checking, manual-review fallback, and confirmed canonical payload
  generation for `DesignService.confirm_translation`.
- `src/adapter/llm/` replaces the placeholder module with:
  - `local.py` deterministic local heuristic translator, the default fallback.
  - `openai.py` opt-in OpenAI Responses API adapter using Structured Outputs
    with model `gpt-5.5`.
  - `anthropic.py` opt-in Anthropic Messages API adapter using forced tool
    schema output with model `claude-sonnet-4-20250514`.
- `tools/ci_gates/llm_output_policy_check.py` is now enforced and checks the
  T-1201 app, adapter, and red-team test surfaces.
