# T-607 Task Brief — `app.decision_tree`

## Scope

Implement the Phase-6 decision-tree application driver for FR-UI-01..12. The service presents guided choices for objective, host, cargo, expression, tagging, cloning chemistry, and biosafety tier, while persisting user progress through the event-sourced `app.design_service` API.

## Inputs

- `CODING_AGENDA.md` v1.5 § 2.6.5.
- `REQUIREMENTS.md` FR-UI-01..12.
- `app.design_service` T-606.
- Part and host catalogue item shapes from T-402/T-403.

## Deliverables

- `src/app/decision_flow.py` typed flow primitives.
- `src/app/decision_tree.py` catalogue-backed decision-tree driver.
- Focused tests for every step's candidate filter.
- Integration test for a complete decision flow that emits design events and produces compile metadata.
