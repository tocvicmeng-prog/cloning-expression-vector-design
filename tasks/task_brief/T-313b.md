# T-313b Task Brief — Production single-writer audit service + IPC

## Scope

Implement the production half of the audit-append split from CODING_AGENDA § 2.3.18:

- SQLite `audit.sqlite` writer owned only by the audit-service process.
- Audit-service handlers for `engine_append` and `admin_append`, authenticated by the production `DecisionRecordVerifier`.
- Framed JSON IPC server and client behind `AuditAppendPort` / `AdminAuditAppendPort`.
- Timeout retry/failure behavior.
- Crash restart verification before accepting new appends.
- Key rotation transition across appends.
- Minimal governance-event writer for `AuditServiceAuthenticationFailed`.
- Deployment runbook.
