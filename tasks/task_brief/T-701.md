# T-701 Task Brief — `engine.codon`

## Scope

Implement the pure Phase-7 codon optimiser. The module owns CAI, MinMax, CHARMING, and avoid-only algorithm selection, lexicographic fixed-point execution with an N=5 cap, and preservation of protected intervals and functional RNA features.

## Inputs

- `CODING_AGENDA.md` v1.5 § 2.7.1.
- `REQUIREMENTS.md` FR-CORE-04..07 and MR-07..09 / MR-17.
- `domain.ports.CodonAlgorithm` contract shape.
- T-601 deterministic adapter behavior as a compatibility reference, without importing adapters into the engine.

## Deliverables

- `src/engine/codon/types.py`
- `src/engine/codon/algorithms.py`
- `src/engine/codon/optimiser.py`
- `src/engine/codon/__init__.py`
- Focused tests for algorithm selection, protein preservation, protected intervals, functional RNA features, motif avoidance, metrics, and invalid input handling.

## Notes

`engine.codon` is intentionally pure and does not import `adapter.biology`. Production or plugin codon adapters can still satisfy `domain.ports.CodonAlgorithm`; the engine package supplies the local deterministic core.
